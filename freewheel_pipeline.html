Pipeline stages
STAGE 01
Freewheel Fetcher
Pull PENDING political ad creatives from Freewheel API
Filter: industries[ ] contains "politic"
Delta cursor — new / updated only
Paginate 100 records per page
Source: Freewheel API
STAGE 02
VAST XML Parser
Enrich each creative with metadata from VAST XML
Fetch vast_content_url per creative
Extract adDescription (adTitle)
Extract fileFormat, fileSize
Source: VAST XML (CDN)
STAGE 03
IronDome Scanner
Submit each creative for spec validation
Send creative_id + vast_url
Returns PASS or FAIL per creative
FAIL → flagged list + GeneralNotes
Source: IronDome API
STAGE 04
ComplyAI Formatter
Map all fields to ComplyAI JSON schema and deliver
Pydantic schema validation
Failed records → dead-letter queue
3 retries with exponential backoff
Source: Freewheel + VAST + IronDome
